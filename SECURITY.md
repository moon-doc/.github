# 🔒 安全政策

## 🚨 报告安全漏洞

如果您发现 Moon-Doc 项目存在安全漏洞，**请勿**在公开的 Issue 中报告。

请通过以下方式负责任地披露：

| 方式 | 说明 |
|------|------|
| 🛡️ **GitHub Security Advisory** | 在对应仓库使用，推荐方式 |
| 📩 **私信组织管理员** | 通过 GitHub 私信联系 |

我们承诺：

| 阶段 | 承诺 |
|------|------|
| ⏱️ 确认收到 | **48 小时内** |
| 📊 初步评估 | **7 天内** |
| 📢 安全公告 | 修复后及时发布并致谢报告者 |

---

## 🛡️ 安全实践

Moon-Doc 项目遵循以下安全实践：

- 🔄 所有依赖定期更新，关注 CVE 通告
- 🔍 CI 中集成安全扫描
- 👀 代码审查关注安全风险
- 🚫 敏感信息（密钥、令牌）绝不提交到代码仓库
